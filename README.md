# Mapping San Diego Bicycle Risk
[Click here to explore the project webpage!](https://agaffney-hub.github.io/MappingSanDiegoBicycleRisk/)

Project with Devon, Sebastian, Isaac and Amy

An interactive web mapping project exploring bicycle safety across San Diego County through fou mapping stories.

## Overview
This project combines collision records, bicycle infrastructure data, weather information, and behavioral factors to identify where cyclists face the greatest risk in San Diego County. The homepage links to four maps that each examine a different dimension of bicycle safety.

## Maps
- **Sebastian** — Infrastructure and collision hotspot analysis
- **Devon** — Weather conditions and crash severity patterns
- **Isaac** — Driver behavior and bicycle safety relationships (map page currently in development / description pending)
- **Amy** — Alcohol-related collision risk

## Data Sources
- SWITRS — California Statewide Integrated Traffic Records System
- City of San Diego Open Data Portal
- SANDAG Regional GIS Data

## Tools Used
- ArcGIS — dashboards and map embedding
- CARTO — map visualization
- GitHub Pages — hosting and deployment
- Leaflet — used in one map page

## More Detailed Summary
For our project, we will look at the relationship between bike paths in San Diego County compared to collisions. By combining infrastructure data of biking paths alongside collision incident data, this project is looking to better understand where and why bicycle related incidents occur and to identify opportunities to improve cyclist safety. Beyond safety concerns, the project highlights the importance of cycling as a transportation option by demonstrating the destinations accessible through San Diego’s bicycle network. This broader context helps illustrate the value of investing in safe, connected and accessible cycling paths. Using bicycle route data provided by SANDAG alongside California Highway Patrol collision incident data, we will investigate if there is an increase in biking incidents along biking routes as more routes are being built across San Diego. Our group will construct a multi-factor spatial analysis of these bicycle collision patterns and explore four different dimensions of risk: infrastructure and collision hotspots, the role of alcohol involvement during incidents, driver aggression and weather conditions. Each factor will be examined through separate maps, allowing us to visualize how different environmental, behavioral and infrastructure related conditions contribute to bicycle safety throughout San Diego County.
